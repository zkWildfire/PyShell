Detailed Description

Base class for commands that run external executables.

PyShell scripts may use this command to run executables for which PyShell does not have native support for. Running a command via this class instead of running it via subprocess will allow PyShell to log the command and handle any errors using the standard PyShell error handling mechanisms.

Validates the arguments for the command.

This method should be overridden by subclasses to validate the arguments for the command. If the arguments are valid, this method should return None. If the arguments are invalid, this method should return a string describing the error.
