Creates a new MultiFileLogger.

Parameters
output_dirDirectory to write log files to. Can be a relative or absolute path. If the path is a relative path, it will be interpreted relative to the directory that the PyShell script is run from.
print_cmd_headerWhether to print command info for each command that is run before the command's output.
print_cmd_footerWhether to print command info for each command that is run after the command's output.
clean_logs_dirWhether to delete any existing log files in the output directory before logging begins.
Exceptions
ValueErrorIf 'output_dir' is a file.
