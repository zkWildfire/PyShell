Detailed Description

Base interface for loggers that handle a single command.

PyShell will request a new command logger instance from the ILogger that the PyShell instance is configured with.

Handles logging for a command being executed.

This method will be invoked repeatedly until the command finishes and all output has been sent to the logger.

Parameters
stdoutThe stdout stream of the command.
stderrThe stderr stream of the command. If the logger requests that the stderr stream be merged with the stdout stream, this will be None.

Logs the results of a command that finished executing.

This method is guaranteed to be executed for every command. Command loggers should handle any cleanup in this method.

Parameters
